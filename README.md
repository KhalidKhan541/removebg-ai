# RemoveBG AI

Free AI-powered background remover that runs entirely in your browser. No uploads, no server - 100% private.

## Features
- One-click background removal
- Replace backgrounds (color, transparent, custom)
- Before/after comparison slider
- Download as PNG/JPG
- History of processed images
- Mobile-friendly

## Tech Stack
- React 18 + TypeScript
- TensorFlow.js + BodyPix
- Tailwind CSS
- Cloudflare Pages

## Live Demo
https://removebg-ai.pages.dev

## Run Locally
```bash
npm install
npm run dev
```

## Deploy
```bash
npm run build
npx wrangler pages deploy dist
```
